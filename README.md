# AMICT CBT Practice

Welcome to the AMICT CBT Practice repository! This project is designed to provide practice tools for Computer-Based Testing (CBT) with a focus on enhancing user experience and effectiveness.

## Installation

To get started with this project, clone the repository:

```bash
git clone https://github.com/themd360/amict-cbt-practice.git
```

Then, navigate to the project directory:

```bash
cd amict-cbt-practice
```

Install the required dependencies:

```bash
npm install
```

## Usage

After installation, you can run the application using:

```bash
npm start
```
